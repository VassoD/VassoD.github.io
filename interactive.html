Interactive images and videos


My selections after using many different platforms:

1.Thinglink


Thinglink allows you to add elements such as a video or hyperlinks to an image that you have downloaded.
Concretely, it is possible to add: a video, an image, a description or a link to a site in the form of "buttons".
The created hotspot is assigned a web address (URL).
When the user goes to this address and hovers his mouse over a button, he sees what has been linked to it appear.
This free Web application therefore makes it possible to make an image "clickable". To do this, it is necessary to register on the site.

2.Genially

Genial.ly is an online tool that can be used to create
animated, or interactive visuals, such as posters, infographics, quizzes, and presentations.
To create visuals, you can use pre-made templates or start from scratch.
Genial.ly can increase engagement in learning and allow students to express their knowledge and develop their communication skills.
Genial.ly is relatively easy to use, especially if you are building on one of the available templates.
When you sign up for Genial.ly, you are provided with a tour of the website, and you can access this tour at any point when using the site.
